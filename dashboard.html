One-time setup (takes ~5 min):
1. Go to script.google.com → New project
2. Paste the Apps Script code (see dashboard-calendar-proxy.gs file)
3. Replace YOUR_ICAL_URL_HERE with your Outlook ICS link
4. Deploy → New deployment → Web app → Anyone can access → Deploy
5. Copy the web app URL and paste it above
Note: Outlook blocks direct browser fetches. Use the Apps Script option above instead.
